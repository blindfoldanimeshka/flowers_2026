# Гид по анимациям в проекте Floramix

## Установленная библиотека
- **framer-motion** - мощная библиотека для создания анимаций в React

## Реализованные анимации

### 1. Главная страница
- **Fade-in анимация** при загрузке страницы
- **Stagger анимация** для категорий (появление с задержкой)
- **Hover эффекты** на карточках категорий (scale)
- **Spring анимации** для карточек товаров

### 2. Каталог товаров
- **Stagger анимация** для списка товаров
- **Hover lift** эффект (поднятие карточки при наведении)
- **Анимированные бейджи** скидок (rotate + scale)
- **Smooth transitions** для счетчика товаров в корзине

### 3. Корзина
- **Slide-in анимации** для элементов корзины
- **Modal анимации** с spring эффектом
- **Exit анимации** при удалении товаров
- **Hover/tap анимации** для кнопок

### 4. Навигация
- **Slide-down анимация** для хедера
- **Fade-in** для информационной панели
- **Spring анимации** для мобильного меню

### 5. Футер и социальные кнопки
- **Fade-in с задержкой** для элементов футера
- **Stagger анимация** для социальных иконок
- **Scale + rotate** анимация при открытии социального меню
- **Hover эффекты** на всех интерактивных элементах

### 6. Переходы между страницами
- **Page transitions** с fade и slide эффектами
- **AnimatePresence** для плавных переходов
- **Exit анимации** при смене роутов

## Утилиты и компоненты

### Готовые варианты анимаций (`/app/client/utils/animations.ts`)
- `fadeInUp` - появление снизу вверх
- `fadeIn` - простое появление
- `scaleIn` - появление с увеличением
- `slideInLeft/Right` - слайд анимации
- `staggerContainer/Item` - для списков
- `bounceIn` - пружинистое появление
- `rotateIn` - появление с вращением

### Компоненты
- `FadeWrapper` - обертка для плавных переходов между страницами
- `AnimatedButton` - кнопка с встроенными анимациями
- `Preloader` - анимированный прелоадер
- `PageTransition` - компонент для анимаций страниц

### Хуки
- `useScrollAnimation` - анимации при скролле
- `useParallaxScroll` - параллакс эффекты

## Примеры использования

### Базовая анимация появления
```jsx
<motion.div
  initial={{ opacity: 0, y: 20 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.5 }}
>
  Контент
</motion.div>
```

### Stagger анимация для списка
```jsx
<motion.div variants={staggerContainer} initial="initial" animate="animate">
  {items.map(item => (
    <motion.div key={item.id} variants={staggerItem}>
      {item.content}
    </motion.div>
  ))}
</motion.div>
```

### Hover эффекты
```jsx
<motion.button
  whileHover={{ scale: 1.05 }}
  whileTap={{ scale: 0.95 }}
>
  Кнопка
</motion.button>
```

## Рекомендации
1. Используйте `AnimatePresence` для exit анимаций
2. Применяйте `motion.lazy` для оптимизации больших списков
3. Используйте готовые варианты из `animations.ts` для консистентности
4. Не перегружайте страницу анимациями - лучше меньше, но качественнее
5. Тестируйте производительность на мобильных устройствах
