# Интеграция с Telegram ботом

## Описание

Telegram бот для отправки уведомлений администраторам о новых заказах в магазине цветов.

## Возможности

- Уведомления о новых заказах в реальном времени
- Доступ к боту только для администраторов с указанным Telegram ID
- Отображение информации о заказе: номер, клиент, позиции, сумма
- Отправка фото первого товара из заказа

## Настройка

### 1. Создание Telegram бота

1. Откройте Telegram и найдите бота [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям для создания бота
4. Скопируйте токен бота (выглядит как `123456789:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 2. Настройка переменных окружения

Добавьте в файл `.env` следующую переменную:

```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
```

### 3. Получение Telegram ID

1. Запустите бота командой:
   ```bash
   npm run bot
   ```

2. Откройте Telegram и найдите вашего бота по имени
3. Отправьте боту команду `/start`
4. Бот отправит вам ваш Telegram ID

Альтернативный способ - используйте бота [@userinfobot](https://t.me/userinfobot)

### 4. Добавление Telegram ID в админ-панель

1. Войдите в админ-панель сайта
2. Перейдите в раздел "Профиль"
3. Вставьте ваш Telegram ID в поле "Telegram ID"
4. Нажмите "Сохранить"

## Запуск

### Запуск бота отдельно

```bash
npm run bot
```

### Запуск вместе с приложением

Для разработки рекомендуется запускать бота в отдельном терминале:

Терминал 1:
```bash
npm run dev
```

Терминал 2:
```bash
npm run bot
```

## Команды бота

- `/start` - Начать работу с ботом и узнать свой Telegram ID
- `/help` - Показать справку по командам
- `/myid` - Показать ваш Telegram ID

## Формат уведомления

При создании нового заказа администраторы с указанным Telegram ID получат сообщение:

```
🆕 Новый заказ #123456

👤 Клиент:
Имя: Иван Иванов
Телефон: +7 (999) 123-45-67
Email: ivan@example.com

📦 Позиции заказа:
1. Букет роз - 2 шт. × 1500 ₽
2. Тюльпаны - 1 шт. × 800 ₽

💰 Итого: 3800 ₽
```

После текстового сообщения бот отправит фото первого товара из заказа.

## Безопасность

- Доступ к уведомлениям имеют только администраторы с указанным Telegram ID
- Если Telegram ID не указан в профиле администратора, уведомления не отправляются
- Бот не имеет других функций кроме отправки уведомлений

## Устранение неполадок

### Бот не отправляет уведомления

1. Проверьте, что `TELEGRAM_BOT_TOKEN` указан в `.env`
2. Убедитесь, что бот запущен (`npm run bot`)
3. Проверьте, что Telegram ID указан в профиле администратора
4. Убедитесь, что вы отправили боту команду `/start` хотя бы один раз

### Ошибка "Forbidden: bot was blocked by the user"

Пользователь заблокировал бота. Разблокируйте бота в Telegram и отправьте `/start`

### Ошибка "Chat not found"

Неверный Telegram ID или пользователь еще не начал диалог с ботом. Отправьте боту `/start`

## Производственное развертывание

Для production рекомендуется использовать процесс-менеджер (например, PM2):

```bash
# Установка PM2
npm install -g pm2

# Запуск приложения
pm2 start npm --name "flowers-app" -- start

# Запуск бота
pm2 start npm --name "telegram-bot" -- run bot

# Сохранение конфигурации
pm2 save

# Автозапуск при перезагрузке
pm2 startup
```
